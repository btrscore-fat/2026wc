# 🏆 2026 美加墨世界杯模拟器

基于 FIFA 排名与球队数据的 48 强完整世界杯模拟器。

## 功能特点

- ✅ 48 支球队完整分组（12组×4队）
- ✅ 小组赛 + 淘汰赛完整模拟
- ✅ 基于 FIFA 排名和胜率的比赛结果计算
- ✅ 加时赛和点球大战支持
- ✅ 实时比分显示
- ✅ 冠军预测

## 赛制说明

- **小组赛**: 12 组，每组 4 队，前 2 名 + 8 个最佳第 3 名晋级
- **淘汰赛**: 32 强 → 16 强 → 8 强 → 4 强 → 决赛
- **比赛场次**: 共 104 场

## 使用方法

1. 点击「开始模拟」按钮
2. 查看小组赛积分榜
3. 查看淘汰赛对战结果
4. 查看最终冠军

## 数据来源

- FIFA 官方排名
- 球队历史胜率
- 球队身价数据
- 核心球员信息
